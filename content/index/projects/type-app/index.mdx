TYPE is an NLP-enhanced writing management web app for ESL classrooms (or alternatively, writing support groups) that offers features like pomodoro timers, outlining tools, named entity recognition, research wizard, spelling suggestions, revision history/source control, classroom forum stream, etc. In doing so, TYPE allows members of classrooms to write more productively, confidently, and calmly.
